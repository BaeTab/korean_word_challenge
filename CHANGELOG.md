# 변경 이력

이 프로젝트의 주요 변경 사항을 기록합니다. [Keep a Changelog](https://keepachangelog.com/ko/1.0.0/) 형식을 따릅니다.

## 2026-07-10

대규모 리디자인. 시즌을 새로 시작하며 게임필·사운드·테마·통계를 전반적으로 다듬었습니다.

### 추가
- **시즌 리셋** — 서버 랭킹/레벨 DB를 전체 초기화하고, 클라이언트도 브라우저당 1회 로컬 기록(개인 통계·업적·데일리 참여 기록)을 함께 초기화해 "내 기록 판수 ≠ 랭킹 판수" 불일치를 방지. 새 시즌 안내 모달 노출.
- **레벨업 오버레이** — 레벨업 시 토스트 한 줄 대신 전용 풀스크린 연출(마스코트 + 컨페티 + Lv.N 표시)로 교체.
- **포인트 획득 플로팅** — 승리로 포인트를 얻으면 헤더 SHOP 스탯 위로 `+N 🪙`가 떠오르는 연출 추가.
- **타일 리빌 플립 애니메이션** — 행 제출 시 타일이 순차적으로 뒤집히며 판정색이 드러나는 연출 추가(`prefers-reduced-motion` 환경에서는 즉시 표시로 대체).
- **키보드 프레스 피드백** — 가상 키 입력 시 눌림 효과(스케일 축소) + 틱 사운드 + 짧은 진동 추가.
- **사운드 & 햅틱** — 오디오 파일 없이 WebAudio로 즉석 생성하는 효과음(키 입력·제출·리빌·승리·패배·레벨업·코인) + 진동 지원. 헤더/인트로 화면에 🔊/🔇 토글 추가(localStorage에 유지).
- **라이트 테마** — 🌙/☀️ 토글로 라이트/다크 테마 전환, 최초 접속 시 시스템 설정(`prefers-color-scheme`)을 자동 감지. 선택값은 localStorage에 유지.
- **오답노트** — 최근 실패한 단어를 최대 20개까지 기록해 프로필 화면에 표시.
- **헷갈린 자모 TOP 5** — 자모별 오답률을 집계해 프로필에서 가장 자주 틀리는 자모 순위 표시(5회 이상 시도한 자모만 집계).

### 수정
- **[버그] 장착한 스킨이 마스코트에 반영되지 않던 문제** — 상점 아이템 ID와 마스코트 스킨 키가 서로 다른 값이라 장착이 화면에 반영되지 않던 것을 매핑 함수(`skinKeyOf`)로 수정.
- **상점 잔액/보유 목록 최신화** — 상점을 열 때마다 서버에서 최신 플레이어 상태를 재조회해 잔액·보유 아이템이 오래된 값으로 보이는 문제 방지.
- **상점 에러 메시지 세분화** — 동기화 문제(`permission-denied`)와 일반 실패를 구분해 안내하도록 개선.
- **7칸 모드 후보 단어 수 표기** — 인트로 화면의 문구를 실제 단어 목록 길이에서 동적으로 계산하도록 수정(하드코딩된 값과의 불일치 방지).

## 2026-07-09

유저 참여형 기능과 상점/레벨 시스템의 기반을 추가한 배포.

### 추가
- **유저 단어 제안** — 다음 시즌 후보 단어를 직접 제안할 수 있는 기능.
- **출석 체크** — KST 기준 하루 1회 체크인 시 XP 보너스 지급, 연속 출석일수 추적 및 관련 업적.
- **상점 & 인벤토리** — 승리로 적립한 포인트로 마스코트 코스메틱 스킨을 구매·장착. 이후 스킨을 6종(고양이·유령·로봇·외계인·판다·닌자)으로 확대하고 인벤토리 구조로 재설계.
- **커스텀 방** — 원하는 단어로 방을 만들어 링크를 공유하고 친구와 같은 문제로 겨루는 기능(방당·닉네임당 1회 제출).
- **레벨 시스템 고도화** — XP 공식 재설계, 업적(배지) 시스템, 프로필 화면 추가.
- **7칸 마스터 모드 단어 확대** — 정답 후보를 42개에서 116개로 확대.

### 수정
- **서버 기록 저장 안정성** — 랭킹/레벨 기록 저장 실패 시 1회 자동 재시도 후, 그래도 실패하면 사용자에게 알림.
